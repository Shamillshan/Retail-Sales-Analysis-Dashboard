 Retail Sales Performance Dashboard
📌 Project Overview
This project analyzes retail sales data to identify key revenue drivers, profitability trends, and product performance across different regions and categories.
An interactive dashboard was developed using Microsoft Power BI to help stakeholders quickly understand business performance and make data-driven decisions.

📂 Dataset

The dataset used for this project is the Sample Superstore Dataset.
It contains 9,994 sales transactions from a retail business. Each record represents a product sold in an order.

Key fields include:
Order Date
Region
Category / Sub-Category
Sales
Profit
Quantity
Discount
Customer Segment

🛠 Tools Used

Microsoft Power BI
Microsoft Excel

📈 Dashboard Features

The dashboard includes the following visualizations:
KPI Cards (Total Sales, Total Profit, Quantity Sold, Profit Margin)
Sales by Region
Monthly Sales Tren
Sales by Category
Discount vs Profit Analysis
Top Products by Sales
